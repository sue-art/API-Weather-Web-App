# API Weather Web App


## Project Summary

This project provides 7-day weather forecasts for major European cities using the 7Timer API, developed with HTML, CSS, JavaScript, and JSON.

## Features
- 7Timer API
- Boostrap
- Swiper Slider
- Skeleton lazy loading 
- Nice Icons
